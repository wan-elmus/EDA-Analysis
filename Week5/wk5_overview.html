Week 5: Data Science Automation

Overview

Almost every specialized field ends up with more and more automation over time, and data science is no different. There are several software tools that have been developed, and more are being developed all the time. We can also automate some of our data science work with Python scripts and other techniques. We will look at some of the automation of data science and how we should position ourselves for the future of data science where more automation will happen.

Learner Outcomes

Upon the completion of this week, you should be able to

  1. Describe the different types of data science automation and data science automation tools.
  2. Use autoML in Python.
  3. Automate part of a data science workflow using a Python script.
  4. Use GitHub to store Python code.
